\- CSS Externo
O CSS externo é quando usamos um arquivo separado e conectamos ao HTML com a tag <link>. Ele permite reutilizar estilos em várias páginas.
\- CSS Interno
O CSS interno é quando usamos a tag "<style>" dentro do arquivo HTML, geralmente dentro da "<head>", ele serve para estilizar apenas aquela página específica.
SELETORES
\- Seletor de elemento
Seleciona todas as tags iguais.
\- Seletor de classe
Seleciona elementos com uma classe específica.
\- Seletor de ID
Seleciona um único elemento com ID.
\- Seletor universal
Seleciona todos os elementos da página.
\- Seletor descendente
Seleciona elementos dentro de outros.
